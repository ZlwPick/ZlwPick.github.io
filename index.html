Redis 笔记1

Redis 是完全开源免费的,遵守BSD协议,是一个高性能(NoSQL)的key-value数据库
NOSQL 与 SQL
数据与数据之间没有关联关系
表与表之间建立关联关系
NOSQL数据库的产生就是为了解决大规模数据集合多重数据种类带来的挑战,尤其是大数据应用的难题.

特点

性能极高 读写速度快

丰富的数据类型

原子性

丰富的特性

redis.conf 配置文件详解

  1. Redis默认不是以守护进程的方式运行,可以通过该配置项修改,使用yes 启用守护进程
    daemonize no

  2. pid 文件的路径
    pidfile /var/run/redis_6379.pid

  3. 端口号
    port 6379

  4. 绑定主机地址
    bind 127.0.01
    本地回环地址。这样的话，访问redis服务只能通过本机的客户端连接，而无法通过远程连接，

  5. 客户端闲置多久后关闭连接
    timeout 300 (0 表示关闭这个功能)

  6. 指定日志记录级别 (debug、verbose、notice、warning)
    loglevel verbose

  7. 日志记录方式,默认标准输出,如果Redis为守护进程方式运行,而这里又配置为日志记录方式为标准输出,则日志发送/dev/null
    logfile stdout

  8. 设置数据库的数量
    databases 16

  9. 指定在多长时间内,有多少次更新操作,就将数据同步到数据文件,可以多个条件配合
    save
    Redis 默认配置文件中提供了三个条件:
    save 900 1 (15分钟内有1个更改)
    save 300 10 (5分钟内有10个更改)
    save 60 10000 (60秒内有10000个更改)

10.指定存储本地数据库时是否压缩数据,
rdbcompression yes

  1. 指定本地数据库文件名,默认值为 dump.rdb,在bin中
    dbfilename dump.rdb

  2. 指定本地数据库存放目录
    dir ./

    requirepass foobared

  2. 设置同一时间最大客户端连接数:默认无限制
    maxclients 128

  3. 指定Redis最大内存限制,
    maxmemory

安装

下载

wget http://download.redis.io/releases/redis-4.0.6.tar.gz

解压

tar -zxvf

yum安装gcc依赖

yum install gcc

跳转到redis解压目录下

cd redis-4.0.6

编译安装

make MALLOC=libc

cd src && make install

Redis 常用命令

Redis 键 key

  1. 删除key的值
    DEL key

  2. 序列化给定key,并返回序列化的值
    dump key

  3. 检查key 是否存在
    exists key

  4. 为给定key 设置过期时间(以秒计)
    expire key seconds

  5. 以秒为单位,返回给定key的剩余生存时间
    ttl key
    pttl key *(毫秒)

5.移除过期时间(永久保持)
persist key

  1. 查找所有符合给定模式(pattern) 的key
    keys pattern
  • 所有
    ？ 代表一个字符

  1. 随机 key
    randomkey

  2. 修改key名
    rename key newkey

9 当前数据库的key移动到给定数据库中
move key db

10.返回key所储存的值的类型
type key

key 命名规范
redis 单个 key 存入512M大小
1.key不要太长,不要超过1024字节
2.key不要太短,key可读性会降低
3.key最好使用统一的命名模式. user:123:password;

set key value (有值就覆盖)

setnx key value (如果key不存在,进行赋值,存在不赋值)

get key
取值

getrange key start end
取字符串中start 到 end 范围的字符

getbit key offset
取指定偏移量上的位

mget key1 key2…
获取一个或者多个key

getset key_name value

Hash 命令

赋值语法：
单个hash
HSET KEY FIELD VALUE
多个hash
HMSET KET FIELD1 VALUE1 [FIELD2 VALUE2]
hmset users uname zlw age 20 address “荆门市”

取值语法
HGET KEY FIELD
HGET KEY FIELD [FIELD1]
HGEALL key 返回hash表中所以字段和值

HKEYS KEY 获取所有哈希表中的字段
HLEN KEY 获取hash表中字段长度

删除语法

HDEL KEY FIELD1 [FIELD2] //删除一个或者多个hash表字段

hsetnx key field value (只有在字段field不存在,设置哈希字段的值)

hincrby key field value (指定字段的整形值加上增量increment)
hincrbyfloat (浮点型)

hexists key field ()查看hash表中，指定的字段是否存在
